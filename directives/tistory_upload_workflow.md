# 티스토리 업로드 워크플로우 (Selenium)

## 개요
생성된 블로그 포스팅을 **Selenium(브라우저 자동화)**을 사용하여 티스토리에 자동으로 업로드하는 워크플로우입니다.
티스토리 API 서비스 종료로 인해, 실제 크롬 브라우저를 띄워 **로그인 -> 글쓰기 -> 발행** 과정을 자동으로 수행합니다.

## 사전 요구사항
1.  **Chrome 브라우저**가 설치되어 있어야 합니다.
2.  `.env` 파일에 티스토리(카카오) 계정 정보가 설정되어 있어야 합니다:
    - `TISTORY_ID`: 카카오 계정 아이디 (이메일)
    - `TISTORY_PW`: 비밀번호
    - `TISTORY_BLOG_NAME`: 블로그 주소 식별자 (예: `myblog.tistory.com` 이면 `myblog`)
3.  **Python 라이브러리 설치**:
    - `selenium`, `webdriver_manager`, `pyperclip`, `python-dotenv`

## 단계별 실행 가이드

### 1단계: 라이브러리 설치
필요한 파이썬 패키지를 설치합니다.
```bash
pip install selenium webdriver_manager pyperclip python-dotenv
```

### 2단계: 업로드할 콘텐츠 준비
`result/` 폴더 내에 생성된 포스팅 결과물이 있어야 합니다. (예: `result/2026-02-08_topic_name`)
- `blog_post.html`: 업로드할 본문 내용
- `hashtags.txt`: 태그 목록
- `*.png`: 본문에 삽입될 이미지 파일들 (예: `image1.png`, `image2.png`)

### 3단계: 업로드 스크립트 실행
아래 명령어를 입력하여 업로드를 시작합니다. 경로(`result/...`)는 실제 폴더명으로 변경해주세요.
```bash
py execution/upload_to_tistory_selenium.py result/2026-02-08_topic_name
```

### 4단계: 진행 상황 모니터링
스크립트가 실행되면 크롬 브라우저가 자동으로 열립니다.
1.  **로그인**: 카카오 로그인 화면이 나타납니다.
    - **주의**: 2단계 인증(2FA)이 설정된 경우, 핸드폰으로 전송된 인증 요청을 **직접 승인**해야 합니다. 스크립트는 로그인이 완료될 때까지 기다립니다.
2.  **글 작성**: 에디터 화면으로 이동하여 제목과 본문(HTML)이 자동으로 입력됩니다.
3.  **발행**: 기본적으로 **'비공개'** 상태로 발행됩니다.
4.  **확인**: 작업이 완료되면 브라우저가 닫힙니다. 티스토리 블로그 관리자 페이지에서 글이 정상적으로 등록되었는지 확인하세요.
